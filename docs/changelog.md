# NexLog Changelog

## 1.0.0

- Rebranded the project to NexLog.
- Added the QML Cyber OS desktop interface with animated startup branding.
- Added local-first investigation dashboard, findings, timeline, MITRE, tools, and 3D-style attack graph views.
- Added release cleanup, release checks, source ZIP packaging, and PyInstaller packaging support.
- Added packaged runtime path handling for QML, assets, detection rules, and user-writable workspaces.
- Standardized public environment variables on `NEXLOG_*`.

## Notes

Generated artifacts, evidence, case databases, local secrets, build outputs, and virtual environments are excluded from release packages.
