# NexLog Privacy

NexLog is local-first. Evidence logs, case databases, exports, and analysis
results stay on the analyst workstation by default.

Optional cloud or third-party integrations can send selected data outside the
device only when a user configures API keys or endpoints. Review provider terms
before enabling AbuseIPDB, Groq, Gemini, Anthropic, VirusTotal, OTX, or similar
services.

Do not upload real evidence to public issue trackers. Redact logs before
sharing bug reports.
