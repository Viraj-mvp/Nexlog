# Third-Party Notices

NexLog depends on open-source libraries. Review each dependency license before
redistributing packaged binaries.

Major dependencies include PySide6, PyInstaller, PyYAML, defusedxml, ijson,
python-evtx, ReportLab, NetworkX, FastAPI, Uvicorn, NumPy, scikit-learn,
sentence-transformers, transformers, tokenizers, PyTorch, and ChromaDB.

Run `python -m pip list --format=freeze` in the release environment to generate
the exact dependency inventory for a published build.
