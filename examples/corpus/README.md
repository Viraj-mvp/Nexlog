# NexLog Parser And Rule Corpus

This folder is the proof corpus for enterprise parser and detection coverage.
Each manifest entry points to a sample log, the formats it should exercise, and
the minimum expected parser/rule behavior.

Add real enterprise samples here before marketing a parser as production-ready:
EVTX/XML, Sysmon, Zeek, Suricata EVE, CloudTrail, IAM, Kubernetes audit,
VPC Flow, DNS, EDR-like JSON, Wazuh, Apache, SSH, and Windows event samples.

Keep samples synthetic, public, or explicitly licensed for redistribution. Do
not commit customer evidence, secrets, real tokens, or private IP inventories.
